```
  Algo: local_maximum[i] = max(A[i], A[i] + local_maximum[i-1])
  
  int maxSumSubArray(int[] A) {
    n = A.length
    local_max = 0
    global_max = Integer.MIN_VALUE
    
    loop 0 ... n-1
      local_max = max(A[i], A[i]+local_max);
      if local_max > global_max
        global_max = local_max
  }
```
